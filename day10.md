learning: Personal Portfolio Website: My Developer Identity, Live on the Web 🚀
Every developer needs a home on the internet. Today I built mine — a fully responsive, modern personal portfolio website that tells my story, showcases my work, and opens doors to new opportunities.
What I Built
🎨 Hero Section with Typing Animation — dynamic role cycling through "Frontend Developer → ECE Engineer → C++ Programmer" with a glowing cyan cursor and smooth entrance animations.
👤 About Me Section — presented as a JSON code block (because why not speak the language?) with my background, interests, and quick.
⚡ Animated Skills Dashboard — scroll-triggered progress bars for HTML5, CSS3, JavaScript, C++, and Responsive Design, plus tool tags and a "currently learning" section.
🗂️ Project Cards — clean cards with tech stack tags, GitHub and live demo links, and hover animations that lift each card off the page.
No frameworks. No build step. Just one clean HTML file — because sometimes the fundamentals are enough.
